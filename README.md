# Translio 🌐

A simple yet powerful artificial intelligence language translation tool that allows users to translate text between multiple languages instantly using modern translation APIs.

🔗 [Live Demo: Translio Website](https://shrutika2026.github.io/CodeAlpha-Translio-Ai-Task1/)

---

## 📖 Project Description
Translio is a web-based application designed to make language translation seamless and user-friendly. It provides an intuitive interface where users can input text, select the source language, choose the target language, and instantly receive translated output.

This project was developed as part of the **CodeAlpha Software Engineering Internship Program**. The app leverages modern translation APIs to ensure accurate translations, while usability features such as copy-to-clipboard and text-to-speech enhance the overall user experience.

---

## ✨ Key Features
* 🌐 **Multi-Language Support** – Translate text seamlessly between a wide range of global languages.
* ⚡ **Real-Time Translation** – Instant, low-latency responses from the translation API.
* 📋 **Copy Button** – Quickly copy translated text to the clipboard with a single click.
* 🔊 **Text-to-Speech** – Listen to the translated text pronunciation for better understanding.
* 📱 **Responsive Design** – Optimized UI that works smoothly across desktop, tablet, and mobile devices.

---

## 🛠️ Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **API Integration:** Google Translate API / Microsoft Translator API

---

## ⚙️ Core Functionality
1. User enters text into the input field.
2. User selects the source and target languages.
3. The text payload is processed via the artificial intelligence translation API.
4. The translated text is dynamically rendered on screen.
5. Users can optionally utilize the copy-to-clipboard or text-to-speech audio playback features.
